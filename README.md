# NoteKeeper App (Assignment 2)

This is an Android application developed as part of the Mobile App Development Lab.

## Features:
- Add notes with title, description, and image
- Store notes using SQLite database
- View notes in list format
- Background task using WorkManager
- Notification: "Review your saved notes today"
- Accelerometer sensor for shake detection

## Database:
- Database Name: NotesDB_18
- Table Name: notes_18
- Extra Field: reminder_flag

## Tech Stack:
- Java
- Android Studio
- SQLite
- WorkManager
